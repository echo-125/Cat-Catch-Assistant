# 猫抓助手 - 油猴脚本使用说明

## 📖 功能介绍

这是一个油猴（Tampermonkey）脚本，用于自动化提取网页中的M3U8视频链接。

### ✨ 主要功能

- ✅ **自动点击播放按钮** - 页面加载后自动点击视频播放按钮
- ✅ **自动跳过广告** - 等待5秒后自动点击"跳過廣告"按钮
- ✅ **智能链接识别** - 自动识别并过滤广告链接
- ✅ **目标链接提取** - 精准提取 `index.jpg` 或 `index.m3u8` 链接
- ✅ **可视化面板** - 显示所有捕获的链接
- ✅ **一键复制** - 点击按钮即可复制链接到剪贴板

## 📦 安装步骤

### 1. 安装 Tampermonkey 扩展

**Chrome 浏览器：**
1. 访问 Chrome 网上应用店
2. 搜索 "Tampermonkey"
3. 点击"添加到 Chrome"
4. 或直接访问：https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo

**Firefox 浏览器：**
1. 访问 Firefox 附加组件商店
2. 搜索 "Tampermonkey"
3. 点击"添加到 Firefox"
4. 或直接访问：https://addons.mozilla.org/firefox/addon/tampermonkey/

**Edge 浏览器：**
1. 访问 Edge 加载项
2. 搜索 "Tampermonkey"
3. 点击"获取"

### 2. 安装脚本

**方法一：从文件安装**
1. 点击浏览器工具栏的 Tampermonkey 图标
2. 选择"添加新脚本"
3. 删除编辑器中的所有内容
4. 将 `m3u8_sniffer_userscript.js` 文件内容复制粘贴到编辑器
5. 按 `Ctrl+S` 保存

**方法二：拖拽安装**
1. 直接将 `m3u8_sniffer_userscript.js` 文件拖拽到浏览器窗口
2. Tampermonkey 会自动识别并提示安装
3. 点击"安装"

### 3. 验证安装

1. 访问任意匹配的网站（如：https://rouva4.xyz/v/*）
2. 点击 Tampermonkey 图标
3. 确认脚本已启用（显示绿色数字）

## 🚀 使用方法

### 基本使用

1. **访问视频页面**
   - 打开支持的视频网站
   - 脚本会自动加载

2. **等待自动操作**
   - 脚本会自动点击播放按钮
   - 等待5秒广告播放
   - 自动点击"跳過廣告"按钮

3. **查看结果面板**
   - 页面右上角会显示结果面板
   - 目标链接（index.jpg）会标记为 ✅
   - 其他视频链接也会显示

4. **复制链接**
   - 点击"复制链接"按钮
   - 链接会自动复制到剪贴板
   - 目标链接会自动复制

5. **使用链接下载**
   - 将复制的链接粘贴到 M3U8 下载器
   - 开始下载视频

### 手动操作

如果自动操作失败，可以手动操作：

1. **手动点击播放**
   - 点击视频中间的播放按钮

2. **手动跳过广告**
   - 等待5秒
   - 点击"跳過廣告 ›"按钮

3. **查看捕获的链接**
   - 结果面板会实时显示所有捕获的链接
   - 目标链接会自动高亮

## ⚙️ 配置选项

脚本顶部有配置项，可以根据需要修改：

```javascript
const CONFIG = {
    autoClickPlay: true,        // 自动点击播放按钮
    autoSkipAd: true,           // 自动跳过广告
    adWaitTime: 5000,           // 广告等待时间（毫秒）
    showResult: true,           // 显示结果面板
    filterKeywords: ['ad', 'ads', 'adv', 'advertisement'], // 广告关键词
};
```

### 配置说明

- **autoClickPlay**: 是否自动点击播放按钮
  - `true`: 自动点击（推荐）
  - `false`: 手动点击

- **autoSkipAd**: 是否自动跳过广告
  - `true`: 自动跳过（推荐）
  - `false`: 手动跳过

- **adWaitTime**: 广告等待时间
  - 默认 5000 毫秒（5秒）
  - 可根据实际情况调整

- **showResult**: 是否显示结果面板
  - `true`: 显示面板（推荐）
  - `false`: 不显示面板

- **filterKeywords**: 广告关键词列表
  - 包含这些关键词的链接会被过滤
  - 可自定义添加

## 🎯 支持的网站

脚本默认支持以下网站：

- `https://rouva4.xyz/v/*`
- `https://*.xyz/v/*`

### 添加新网站

如果需要在其他网站使用，修改脚本头部的 `@match` 规则：

```javascript
// @match        https://example.com/video/*
// @match        https://*.example.com/*
```

## 🔍 识别规则

### 目标链接识别

脚本会识别以下链接为目标链接：

- 包含 `index.jpg`
- 包含 `index.m3u8`

这些链接会被标记为 ✅ 并自动复制。

### 广告链接过滤

脚本会过滤包含以下关键词的链接：

- `ad`
- `ads`
- `adv`
- `advertisement`

这些链接不会显示在结果面板中。

## ❓ 常见问题

### 1. 脚本没有运行？

**解决方法：**
- 确认 Tampermonkey 已安装并启用
- 确认脚本已启用（Tampermonkey 图标显示绿色数字）
- 检查网站是否匹配 `@match` 规则
- 刷新页面重试

### 2. 没有找到播放按钮？

**解决方法：**
- 检查页面是否完全加载
- 尝试手动点击播放按钮
- 查看控制台日志

### 3. 没有捕获到链接？

**解决方法：**
- 确认视频已开始播放
- 检查网络请求拦截是否启用
- 查看控制台是否有错误信息

### 4. 找到了多个链接？

**解决方法：**
- 目标链接（index.jpg）会标记为 ✅
- 其他链接可能是广告或预览视频
- 优先使用标记为目标链接的URL

### 5. 链接复制失败？

**解决方法：**
- 手动点击"复制链接"按钮
- 或在控制台查看链接并手动复制
- 检查浏览器剪贴板权限
